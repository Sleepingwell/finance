This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts. Calculates the internal rate of return for a series of payments using Newton's method.

The algorithm starts with an initial estimate of 0.0, and runs until the result is either within tolerance of the 'true' root, or for a maximum of maxIterations iterations.

Template Parameters:
IterPThe type of the iterators over the payments.
IterTThe type of the iterator over the time periods.
Parameters:
beginpIterator pointing to the beginning of the payments.
endpIterator pinting to the end of the payments.
begintIterator pointing the beginning of the times of the payments.
endtIterator pointing to the end of the times of the payments. If there are more time points than payments, then the number of payments is reduced to match the number of time periods.
toleranceSpecifies how 'exact' the answer should be (the answer will be within tolerance of the true value).
maxIterationsThe maximum number of iterations to run.

Calculates the internal rate of return for a series of payments using Newton's method.

The algorithm starts with an initial estimate of 0.0, and runs until the result is either within tolerance of the 'true' root, or for a maximum of maxIterations iterations.

Parameters:
beginpIterator pointing to the beginning of the payments.

This version assumes that the payments are one period appart.
